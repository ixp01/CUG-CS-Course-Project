# Sutherland-Hodgeman 多边形裁剪算法 - 使用说明

## 🎮 交互式步骤演示（重点功能！）

本程序实现了**交互式裁剪步骤演示**，可以通过键盘控制逐步查看裁剪过程！

## 🪟 三个窗口说明

程序运行后会打开3个窗口：

### 窗口1 - 原始图形
- **显示内容**：原始多边形（淡蓝色）+ 裁剪窗口（红色边框）
- **用途**：展示待裁剪的原始数据

### 窗口2 - 裁剪后的图形
- **显示内容**：最终裁剪结果（淡绿色）
- **用途**：展示完整裁剪后的最终结果

### 窗口3 - 交互式步骤演示 ⭐⭐⭐
- **显示内容**：可交互的裁剪过程演示
- **特点**：
  - 从步骤0（原始多边形）到步骤4（最终结果）
  - 每个步骤使用不同的颜色
  - 显示当前顶点数量和坐标
  - 带有进度条和操作提示

## ⌨️ 键盘控制（在窗口3中使用）

### 基本操作
| 按键 | 功能 | 说明 |
|:---:|:---|:---|
| **空格键** | 下一步 | 进入下一个裁剪步骤 |
| **回车键** | 下一步 | 与空格键功能相同 |
| **退格键** | 上一步 | 返回到上一个步骤 |

### 快速跳转
| 按键 | 跳转到 |
|:---:|:---|
| **0** | 步骤0 - 原始多边形 |
| **1** | 步骤1 - 左边界裁剪后 |
| **2** | 步骤2 - 下边界裁剪后 |
| **3** | 步骤3 - 右边界裁剪后 |
| **4** | 步骤4 - 上边界裁剪后（最终结果）|

### 其他功能
| 按键 | 功能 |
|:---:|:---|
| **R** | 重置到步骤0 |
| **H** | 在控制台显示帮助信息 |
| **ESC** | 退出程序 |

## 📊 裁剪步骤详解

### 步骤 0：原始多边形
- **颜色**：淡蓝色
- **说明**：显示原始的7个顶点的多边形
- **顶点数**：7

### 步骤 1：左边界裁剪 (x ≥ 100)
- **颜色**：浅橙色
- **说明**：用裁剪窗口的左边界进行第一次裁剪
- **裁剪规则**：保留 x ≥ 100 的部分

### 步骤 2：下边界裁剪 (y ≥ 100)
- **颜色**：浅黄色
- **说明**：在步骤1的基础上，用下边界继续裁剪
- **裁剪规则**：保留 y ≥ 100 的部分

### 步骤 3：右边界裁剪 (x ≤ 300)
- **颜色**：浅紫色
- **说明**：在步骤2的基础上，用右边界继续裁剪
- **裁剪规则**：保留 x ≤ 300 的部分

### 步骤 4：上边界裁剪 (y ≤ 300)
- **颜色**：浅青色
- **说明**：最后一次裁剪，得到最终结果
- **裁剪规则**：保留 y ≤ 300 的部分

## 🎨 视觉效果说明

### 界面元素
- **裁剪窗口**：红色粗边框（始终显示）
- **多边形填充**：每个步骤使用不同的柔和颜色
- **多边形轮廓**：深灰色边框
- **顶点标记**：红色圆点
- **顶点标签**：显示顶点编号和坐标 `P0(x,y)`

### 信息显示
- **顶部**：当前步骤标题和描述
- **左下角**：键盘控制说明
- **右上角**：进度条（显示当前步骤/总步骤）
- **顶点标注**：每个顶点旁边显示编号和坐标

## 🚀 使用流程

1. **启动程序**
   - 运行后会打开3个窗口
   - 控制台会显示裁剪过程的详细数据

2. **聚焦到窗口3**
   - 点击窗口3使其获得焦点
   - 此时窗口3显示"步骤0：原始多边形"

3. **逐步演示**
   - 按**空格键**进入下一步
   - 观察多边形的变化
   - 注意颜色的改变和顶点数量的变化

4. **重复演示**
   - 按**R键**重置到开始
   - 或按**0**直接跳到步骤0
   - 再次按空格键重新演示

5. **截图保存**
   - 在每个步骤按截图键
   - 保存到实验报告中

## 📝 演示建议

### 给老师/同学演示时
1. 先展示窗口1：介绍原始多边形和裁剪窗口
2. 展示窗口2：说明最终裁剪结果
3. **重点展示窗口3**：
   - 从步骤0开始
   - 逐步按空格键，每一步都解释：
     - 当前裁剪的是哪条边界
     - 裁剪规则是什么
     - 顶点数量的变化
     - 为什么会产生新的交点
   - 演示可以返回上一步（退格键）
   - 演示直接跳转（数字键）

### 制作实验报告时
1. 在每个步骤截图（0-4，共5张图）
2. 控制台输出也截图（显示详细顶点坐标）
3. 在报告中按步骤逐一分析
4. 配合代码说明算法实现

## 💡 技巧提示

1. **多次演示**：可以反复按空格键和退格键，前后对比观察变化
2. **快速定位**：想看某一步可以直接按对应数字键

## ❓ 常见问题

**Q: 为什么按键没反应？**  
A: 确保点击了窗口3，使其获得焦点（窗口标题栏会高亮）

**Q: 如何重新开始演示？**  
A: 按 R 键或数字键 0 都可以重置到初始状态

**Q: 每个步骤的顶点坐标在哪里看？**  
A: 
- 图形上：每个顶点旁边有标注 P0(x,y)
- 控制台：程序启动时已经打印所有详细数据

**Q: 能否自动播放？**  
A: 当前版本需要手动按键，这样更方便演示和讲解

**Q: 如何保存截图？**  
A: 
- Windows: 使用 Win+Shift+S 或 PrintScreen 键
- 或使用截图工具（如 Snipping Tool）
