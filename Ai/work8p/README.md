# 基于LeNet-5的手写数字识别系统

这是一个使用LeNet-5卷积神经网络架构实现的手写数字识别系统，包含图形用户界面，可以进行数字绘制和实时识别。

## 功能特点

- 使用LeNet-5卷积神经网络识别手写数字（0-9）
- 基于MNIST数据集训练模型
- 提供用户友好的图形界面
- 实时手写数字识别
- 可视化训练过程和识别结果
- 支持模型保存和加载

## 安装依赖

在运行此应用程序之前，请确保安装了以下Python库：

```bash
pip install tensorflow numpy matplotlib pillow
```

## 使用说明

1. 运行应用程序：

```bash
python digit_recognition.py
```

2. 界面分为左右两部分：
   - 左侧：模型训练区域
   - 右侧：手写数字识别区域

3. 模型训练步骤：
   - 点击"加载数据"按钮加载MNIST数据集
   - 设置训练参数（批量大小、Epochs、学习率）
   - 点击"训练模型"按钮开始训练
   - 训练过程中会显示实时进度和训练曲线

4. 手写数字识别：
   - 在右侧黑色画布上用鼠标绘制数字
   - 点击"识别"按钮进行识别
   - 结果会显示预测的数字和置信度
   - 可以点击"清除"按钮清除画布

5. 模型保存与加载：
   - 训练完成后，可以点击"保存模型"按钮保存训练好的模型
   - 也可以通过"加载模型"按钮加载已保存的模型，无需重新训练

## LeNet-5架构

该应用使用了LeNet-5卷积神经网络架构，包含以下层：

- 输入层: 28x28x1
- 卷积层 C1: 6个5x5卷积核
- 池化层 S2: 2x2最大池化
- 卷积层 C3: 16个5x5卷积核
- 池化层 S4: 2x2最大池化
- 全连接层 F5: 120个神经元
- 全连接层 F6: 84个神经元
- 输出层: 10个神经元（对应0-9十个数字）
