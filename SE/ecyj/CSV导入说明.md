# CSV测试数据导入指南

## 📁 测试文件清单

项目文件夹中包含以下4个CSV测试文件：

1. **test_manufacturers.csv** - 厂商信息（5条记录）
2. **test_equipment.csv** - 设备信息（10条记录）
3. **test_records.csv** - 检测记录（13条记录）
4. **test_result_items.csv** - 检测结果项（52条记录，每个检测记录包含PT1/PT2/CT1/CT2四个项目）

## 📋 导入步骤（按顺序）

### 第一步：导入厂商信息
1. 启动系统
2. 点击"厂商信息"标签页
3. 点击工具栏的"导入"按钮
4. 选择 `test_manufacturers.csv`
5. 等待提示"成功导入5条厂商记录"

### 第二步：导入设备信息
1. 点击"设备信息"标签页
2. 点击"导入"按钮
3. 选择 `test_equipment.csv`
4. 等待提示"成功导入10条设备记录"

### 第三步：导入检测记录
1. 点击"检测记录"标签页
2. 点击"导入"按钮
3. 选择 `test_records.csv`
4. 等待提示"成功导入13条检测记录"

### 第四步：导入检测结果项
1. 点击"检测结果项"标签页
2. 点击"导入"按钮
3. 选择 `test_result_items.csv`
4. 等待提示"成功导入52条检测结果项"

## ⚠️ 重要提示

1. **必须按顺序导入**：因为存在外键依赖关系
   - 设备信息依赖厂商信息
   - 检测记录依赖设备信息
   - 检测结果项依赖检测记录

2. **数据关联**：
   - test_records.csv 中的设备ID（1-10）对应 test_equipment.csv 中的设备
   - test_result_items.csv 中的检测记录ID（1-13）对应 test_records.csv 中的记录

3. **检测结果说明**：
   - 每条检测记录包含4个检测项目：PT1、PT2、CT1、CT2
   - PT1/CT1：低负载测试（20%、5%）
   - PT2/CT2：满负载测试（100%）
   - 所有测试数据都在合格范围内

## 🎯 导入后的数据

导入完成后，系统将包含：
- 5家厂商
- 10台设备
- 13次检测记录
- 52个检测结果项（13条记录 × 4个项目）

可以在各个页面查看数据，也可以：
- 选择检测记录，点击"查看详细数据"添加详细测量数据
- 选择检测记录，导出PDF报告
- 使用统计功能查看分析结果
