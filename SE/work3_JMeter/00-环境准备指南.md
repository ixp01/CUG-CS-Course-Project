# ASP.NET Core 开发环境准备指南

## ⚠️ 重要：首先需要安装 .NET SDK

你的电脑目前还没有安装 .NET SDK，这是开发 ASP.NET Core 应用的必备工具。

---

## 📥 步骤 1：下载并安装 .NET 8 SDK

### 方法 1：通过官网下载（推荐）

1. **访问下载页面**：
   - 打开浏览器，访问：https://dotnet.microsoft.com/zh-cn/download/dotnet/8.0
   
2. **选择正确的版本**：
   - 找到 **.NET 8.0 SDK** 部分（不是 Runtime）
   - 根据你的系统选择：
     - **Windows x64**（大多数 Windows 电脑）
     - Windows x86（32位系统，较少见）
     - Windows Arm64（仅限 ARM 架构电脑）

3. **安装**：
   - 下载完成后，双击运行安装程序
   - 点击"安装"并等待完成
   - **安装完成后，重启 VS Code 和终端**

### 方法 2：通过 WinGet（Windows 10/11 自带的包管理器）

如果你的 Windows 版本支持 WinGet，可以在 PowerShell 中运行：

```powershell
winget install Microsoft.DotNet.SDK.8
```

---

## ✅ 步骤 2：验证安装

安装完成并**重启终端**后，在 PowerShell 中运行：

```powershell
dotnet --version
```

**预期输出示例**：
```
8.0.404
```

如果显示版本号（8.x.x），说明安装成功！

---

## 🔧 步骤 3：安装 VS Code 扩展

在 VS Code 中安装以下扩展（必须）：

1. **C# Dev Kit**（微软官方）
   - 提供 C# 语法高亮、智能提示、调试功能
   - 在 VS Code 扩展商店搜索 `C# Dev Kit` 并安装

2. **可选但推荐的扩展**：
   - **C#**（微软官方，通常随 C# Dev Kit 自动安装）
   - **NuGet Gallery**（方便管理 NuGet 包）
   - **REST Client** 或 **Thunder Client**（用于测试 API）

---

## 🗄️ 步骤 4：确认 SQL Server 可用

根据你的实验要求，需要使用 SQL Server。有以下几种选择：

### 选项 1：使用 SQL Server LocalDB（推荐，最简单）

LocalDB 是轻量级的 SQL Server，通常随 Visual Studio 自动安装。

**检查是否已安装**：
```powershell
SqlLocalDB info
```

**如果显示版本信息**（如 `MSSQLLocalDB`），说明已安装，可直接使用！

**如果未安装**，下载并安装：
- 下载页面：https://learn.microsoft.com/zh-cn/sql/database-engine/configure-windows/sql-server-express-localdb
- 选择 **SQL Server Express LocalDB**

### 选项 2：使用完整版 SQL Server Express（免费）

如果你需要完整的 SQL Server 功能：
- 下载：https://www.microsoft.com/zh-cn/sql-server/sql-server-downloads
- 选择 **Express 版本**（免费）

### 选项 3：使用 Docker 运行 SQL Server（适合熟悉 Docker 的同学）

```powershell
docker run -e "ACCEPT_EULA=Y" -e "SA_PASSWORD=YourStrong@Passw0rd" -p 1433:1433 --name sqlserver -d mcr.microsoft.com/mssql/server:2022-latest
```

## 🆘 常见问题

### Q1：安装后终端还是提示"dotnet 不是内部或外部命令"？
**解决方案**：
1. 完全关闭 VS Code
2. 关闭所有 PowerShell 窗口
3. 重新打开 VS Code
4. 打开新的终端再次测试

### Q2：我不想安装完整的 SQL Server，LocalDB 够用吗？
**回答**：够用！LocalDB 完全满足你的作业要求，连接字符串使用：
```
Server=(localdb)\\mssqllocaldb;Database=StudentDB;Trusted_Connection=True;
```

### Q3：我已经有 Visual Studio，还需要安装这些吗？
**回答**：如果你已经安装了 Visual Studio 2022，通常 .NET SDK 和 LocalDB 都已经有了。只需在 VS Code 中安装 C# Dev Kit 扩展即可。
