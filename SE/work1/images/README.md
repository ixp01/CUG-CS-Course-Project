# 📁 图片存放目录

## 说明

请将您的测试图片放在这个目录中。

## 支持的图片格式

- `.jpg` / `.jpeg`
- `.png`
- `.gif`
- `.bmp`

## 使用方法

1. 将至少3-5张图片复制到此目录
2. 图片可以是任意内容（风景、人物、物品等）
3. 建议图片不要太大（每张不超过5MB）

## 示例

```
images/
├── photo1.jpg
├── photo2.jpg
├── photo3.png
└── scenery.jpg
```

## 注意事项

- 图片文件名可以是任意名称
- 服务会自动识别并随机选择
- 不需要修改代码，添加图片后直接可用
